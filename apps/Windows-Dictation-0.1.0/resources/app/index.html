Windows Dictation

Push-to-Talk Diktat für Windows

Einfach Hotkey drücken, sprechen, loslassen – der Text landet direkt im aktiven Programm.

Bereit
Drücke F8, um die Aufnahme zu starten.

Einstellungen

Den Schlüssel bekommst du hier: console.groq.com/keys

Beispiele: F8, F9, Alt+Space

Zuletzt erkannt

Noch nichts transkribiert.

So benutzt du es

  1. Windows Dictation starten
  2. Einmal den Groq-Schlüssel eintragen und speichern
  3. In Word, Mail oder Chat-Fenster klicken
  4. F8 drücken, sprechen, wieder F8 drücken
  5. Der Text wird eingefügt – oder wenigstens in die Zwischenablage kopiert
